#Проект для турниров по голодным играм в Minecraft.
Создаётся мир, на определённое кол-во человек, без каких либо ресурсов и сундуков (кроме штатных строений майнкрафта), куда подключается множество игроков, спавн которых рандомизируется (обязательно на поверхности, не в лаве, воде, пещере и воздухе), смотрители могут наблюдать за каджым, зайдя на сайт, на котором пользователям будет видна карта, с расположением всех игроков. Если нажать по одному из игроков, откроется стрим от его лица, будет виден его инвентарь в реальном времени.
