# Проект для администрации турниров по голодным играм в Minecraft.
## Правила голодных игр:
1. Мир ограничен баръером так, что во все стороны от центра мира будет доступно 10.000 блоков.
2. Баръер со временем сужается (временные рамки выбираются участниками)
3. Вход в некоторые из измерений запрещён в первое время (выбор временных рамок и запрещённых измерений выбирается игроками)
4. PVP между игроками запрещено системой на начальных этапах игры (временные рамки выбирают игроки. К примеру: PVP разрешено с открытием первого измерения/с первым добытым алмазом)
5. Запрещено:
     - Использовать баги и ошибки игры для получения преимущества над игроками
     - Использовать программы, дающие преимущества над игроками
     - Оскорблять других игроков

## Суть проекта
Создаётся мир, на определённое кол-во человек, без каких либо ресурсов и сундуков (кроме базовых строений майнкрафта), куда подключается некое количество игроков, спавн которых рандомизируется (обязательно на поверхности, не в лаве, воде, пещере и воздухе), администраторы могут наблюдать за каджым, зайдя на сайт, на котором пользователям будет видна 2D карта, с расположением всех игроков. Если нажать по одному из игроков, откроется стрим от его лица, будет виден его инвентарь в реальном времени. Администратор сможет отправить в бан, кик, мут игрока и иные действия над игроком и миром. Доступен список всех онлайн и оффлайн игроков, а так же их места спавна, к которым можно будет телепортироваться.
